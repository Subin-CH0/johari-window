조하리의 창(Johari's Window)

본인이 고른 6개 형용사와 타인이 고른 6개 형용사를 비교해 열린 창, 보이지 않는 창, 숨겨진 창, 미지의 창을 자동으로 정리합니다.

본인 선택

0 / 6 선택

아래 56개 형용사 중 본인을 잘 나타내는 형용사 6개를 클릭하세요. 다시 클릭하면 선택이 해제됩니다. 선택 개수는 이동 버튼을 누를 때 확인합니다.

내가 선택한 형용사

아직 선택한 형용사가 없습니다.

타인 선택

0 / 6 선택

응답자는 자신의 이름과 평가 대상의 이름을 입력한 뒤, 그 사람을 잘 나타낸다고 생각하는 형용사 6개를 선택하세요. 선택 개수는 응답 데이터 만들기 버튼을 누를 때 확인합니다.

타인 이름을 입력하면 선택 문구가 표시됩니다.

선택한 형용사

아직 선택한 형용사가 없습니다.

복사해서 전달할 응답 데이터

아래 내용을 복사해 결과를 확인할 사람에게 보내세요. 이름과 선택한 형용사 6개가 함께 들어 있습니다.

결과 보기

타인 응답 0명

타인이 보내준 응답 데이터를 붙여 넣으면, 각 사람이 누구에 대해 어떤 형용사를 골랐는지 모아보고 결과를 계산할 수 있습니다.

열린 창

나도 알고 타인도 아는 특성입니다. 자기인식과 타인의 인식이 만나는 영역입니다.

보이지 않는 창

나는 선택하지 않았지만 타인은 알아본 특성입니다. 피드백을 통해 발견할 수 있습니다.

숨겨진 창

나는 알고 있지만 타인에게는 덜 드러난 특성입니다. 자기노출과 관련됩니다.

미지의 창

나도 타인도 선택하지 않은 특성입니다. 아직 탐색되지 않았거나 해당성이 낮은 영역입니다.

응답 데이터 붙여넣기

가져온 응답

본인 선택과 타인 응답을 준비하세요.
